# 卷一 1.8：2001 年的盘山路（卷一末高潮）—— 分场景大纲

> 字数预算：6000（关键场景章，上限 8000）
> POV：远舟（100%，15 岁实岁）
> 时点：2001 年 7 月（中考榜揭晓 → 申奥之夜 → 周末独自上黄山）
> 地点：屯溪二中 → 屯溪老街 → 林家 → 黄山盘山公路
> 意象触发：父亲的烟（一笔）/ 音乐（周杰伦《范特西》7 月发）/ **云海（出现 1，全卷高潮）**

## 场景（5 个）

### 场景 1：中考揭榜（1000 字）
- 2001 年 7 月初某日早晨
- 屯溪二中校门口贴红榜
- 远舟一个人骑车去。马骏没来（他没考好）
- 红榜上初三毕业生按片区第一名一直排到 200 名
- 远舟在第一行看见自己的名字
  - 屯溪片区第一名：林远舟，652 分
- 旁边几个家长围着看，叫"林家儿子！"
- 远舟没接。把书包带子提一下，转身走
- 校门口周老师正好过来，看见远舟，拍了一下他的肩
  - "你做最后三题做得不错。"
  - 远舟："嗯。"
- 周老师："你想考的是屯溪一中？"
  - 远舟："嗯。"
- 周老师："你能去合肥一中。"
  - 远舟："屯溪一中。"
- 周老师："为啥。"
  - 远舟："我妈想我留屯溪。"
- 周老师没接

### 场景 2：父亲穿白衬衫陪去屯溪一中报到（1200 字）
- 7 月某天，远舟去屯溪一中报到
- 父亲早上六点起来熨白衬衫。母亲帮熨
- 父亲："新衬衫，今天穿。"
- 母亲："新的是去年单位发的。"
- 父亲："还没穿过。"
- 远舟在房间里听见这段
- 七点半父子俩出门
- 父亲走前头，远舟跟后头。父亲今天没骑车，俩人走
- 屯溪一中在长干路。走 25 分钟
- 报到处在体育馆。远舟交材料、领新校服（绿白款）
- 校长在大厅。校长看见父亲："老林。"
  - 父亲："王校长。"
- 校长跟远舟握手："屯溪片区第一名。"
  - 远舟："嗯。"
- 父亲全程没说话。最后父亲从西裤口袋掏一根红梅出来，没点，搁在嘴上含着。父亲在校园里不点烟
- 出校门后父亲点了火
- "你考得好。"
- 父亲就说这五个字
- 远舟："嗯。"
- 俩人走回家

### 场景 3：申奥之夜 · 跑老街（900 字）
- 7 月 13 日（周五）晚 10 点
- 莫斯科萨马兰奇宣布 2008 年奥运会在北京
- 屯溪老街上鞭炮起来
- 远舟和马骏在老街口的网吧里看。网吧 8 元包夜，他俩本来在打 CS
- 萨马兰奇说出"Beijing"那一刻，网吧里有几个人叫
- 远舟和马骏出网吧，跟着鞭炮声跑。从老街口跑到老街尾，一里
- 老街上人多。鞭炮味重
- 跑到老街尾远舟才停下，撑膝盖喘气
- 马骏："牛逼啊！"
- 远舟没接。他看老街尽头新安江。江面有月光
- 末段：远舟跑回家。母亲在客厅没睡，看电视上的转播。母亲："这下你看见了。"远舟："嗯。"

### 场景 4：周末一个人上黄山（1100 字）
- 7 月某周六
- 远舟跟母亲说去汤口同学家。母亲："小心。"父亲："你去吧。"
- 远舟骑自行车从屯溪出发。到汤口 70 公里。早六点出发，路上吃干粮，下午一点到汤口
- 在汤口找了个山脚下旅馆寄了车（5 元）
- 下午两点开始爬。北海方向。
- 爬到光明顶附近是傍晚六点
- 光明顶人不多。两三个挑山工歇着
- 远舟站光明顶。山下云在动
- 他在背包里摸出一瓶矿泉水，喝。再坐了一会儿
- 决定不住山上。回程

### 场景 5：盘山路冲下 + 云海 + 「我无所不能」（1800 字）
- 决定下山。
- 走盘山公路下山段（实际盘山公路当时双向单车道，路边偶有挑山工）
- 远舟从汤口取回自行车
- 时间还早。日头刚过山头。盘山公路下山段坡陡
- 远舟骑车上盘山公路（汤口往上短一段），到一段长下坡起点（坐索）
- ——其实我应该让他从光明顶下山走步行道到汤口，取车之后骑下盘山公路一段
- 远舟骑车冲下盘山公路的下山段。坡陡，弯多
- 他不刹车（其实手指还是搭在刹车上，但没用力，是一种"假装不刹"的少年姿态）
- 山风穿过校服衬衫
- 第三个弯他过完，云海从山谷里翻上来
- 远舟大喊一声："我——无所不能！"
- 山谷里回声。"——能——能——能"
- 山风穿过他的衬衫。他没回头
- 末段：远舟下到山脚汤口。停车。看了一眼天。日头落了。云海散了一半。他骑车继续往屯溪方向走。**最后一句**：「他没回头。」

## 自检
- 视角 100% 远舟
- 关键场景章，字数可超
- 父亲「你考得好」五个字 + 申奥之夜「这下你看见了」母亲一句，给少年得志的高峰
- 云海初次出现，要写到云从山谷里翻上来 ✓
- 「我无所不能」是 spec 锁定的话 ✓
- D1 校霸气：本章是 D1 校霸气的高峰呈现——骑车冲下盘山路 + 大喊「我无所不能」
- 不要写「他感到自由」「他感觉无敌」——只写动作 + 一句喊话
