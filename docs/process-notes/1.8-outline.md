# 1.8 2001 中考 + 盘山路冲下（卷一末高潮）

**POV**：远舟
**字数预算**：6500
**时代标记**：911（远景）、申奥成功
**意象触发**：无（卷一不设意象触发点）

## 关键提示

- 盘山路冲下 = 少年远舟的"自由感"巅峰
- 为后续反复失去自由感做对照
- 情绪外化：不写"他很自由"，写风灌进衣服、什么也不想
- 马骏的反应：害怕、不敢下——对比远舟的果断
- 时代细节：2001年中考、合肥一中、申奥成功（7月）

## 风格要点

- 远舟视角更冷、更观察性
- 盘山路场景要写得有速度感——短句、快节奏
- 情绪克制——不写"兴奋"，写他的行为——直接冲下去
- 这是卷一的高潮，要写出少年感
