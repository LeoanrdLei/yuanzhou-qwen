# 序章 S1 分场景大纲

## 章节信息
- 章节：序章 S1
- 标题：2026 年深秋的阳台
- POV：远舟
- 字数预算：8000
- 时间：2026 年深秋，某周六下午→深夜
- 地点：杭州某公寓阳台

## 场景拆分

### S1-a（~2000字）
- 地点：阳台
- 人物：远舟（苏敏在屋里，只闻声）
- 动作：远舟独自站在阳台抽烟。深秋，风凉。屋里传来苏敏的声音——女儿满月宴的宾客名单要确认。远舟应了一声，没动。他看着楼下的小区，有人在遛狗。
- 意象：无（序章不设意象触发，只做氛围铺垫）
- 时代标记：微信消息提示音、满月宴、杭州小区

### S1-b（~1500字）
- 地点：阳台（闪回切入）
- 人物：远舟（16岁的脸）
- 动作：手机屏幕灭了，黑屏映出他的脸。那张脸和另一张脸重叠——2002年，黄山，某条街上，一个穿紫色羽绒服的女孩走过去。不展开，只给一个画面碎片。然后切回。
- 关键：不写"他想起了"——用物理动作过渡（黑屏→映脸→另一张脸）

### S1-c（~2000字）
- 地点：屋内
- 人物：远舟、苏敏（出场但不给内心）、女儿（睡着）
- 动作：苏敏又喊了一声。远舟掐了烟进去。客厅里苏敏在 pad 上排座位。远舟看了一眼，说了句"你定就好"。苏敏看了他一眼，没说什么。远舟路过女儿房间，推门看了一眼——婴儿床里，女儿睡着了，手攥着。
- 关键：苏敏的"未明朗化"——她出场了，但读者只通过远舟的眼睛看她。不写她怎么想。

### S1-d（~2500字）
- 地点：卧室/客厅
- 人物：远舟
- 动作：深夜。苏敏和女儿都睡了。远舟坐在客厅沙发上，打开手机。微信。翻到一个对话框——最后一条消息是三年前。"新年快乐"。对方头像没换——是一棵树（或别的什么）。他打了几个字。看了几秒。删掉。锁屏。把手机扣在沙发上。
- 关键：不揭穿对方是谁。读者自己判断。

## 意象触发
- 云海：不直接出现，但阳台望出去的天际线可以暗示"远方"
- 紫色：S1-b 闪回中的紫色羽绒服碎片
- 梧桐：不出现（上海意象，序章不涉及）

## 时代标记
- 2026年：微信、pad、杭州互联网圈的生活状态
- 满月宴：中国传统习俗
