# 1.2 分场景大纲

## 章节信息
- 章节：1.2
- 标题：物理竞赛省一
- POV：远舟
- 字数预算：5500
- 时间：1999 年春（3-4月）
- 地点：学校、省城合肥、家

## 场景拆分

### a（~1500字）
- 地点：教室/实验室
- 人物：远舟、物理老师老周
- 动作：物理老师老周给远舟开小灶，准备省竞赛。远舟解题快，但不爱说话。老周给他额外题目，他很快就做完了。老周说"你去合肥考，没问题"。
- 时代标记：物理竞赛热（90年代末保送加分）、手写解题过程
- 关键：建立远舟"天赋型"形象，老周是伯乐

### b（~1500字）
- 地点：省城合肥考场
- 人物：远舟
- 动作：去合肥考试。第一次出黄山。坐大巴4小时。路过长江。到合肥，看到大城市，没说什么。考试。题目不难。考完在街上走了一圈，看到新华书店。
- 时代标记：长途大巴、BP机开始退场（路边公用电话）
- 关键：第一次"出远门"，但远舟反应平淡——他的世界在内心

### c（~1500字）
- 地点：家
- 人物：远舟、父母
- 动作：成绩出来——省一等奖。母亲高兴，逢人就说。父亲说"嗯"，晚饭加了个菜（买了条鱼）。远舟没什么反应。
- 时代标记：成绩公布 via 学校通知/报纸
- 关键：母亲的骄傲 vs 父亲的沉默 vs 远舟的平淡

### d（~1000字）
- 地点：学校
- 人物：远舟、同学、马骏
- 动作：回到学校。同学看他眼神变了——有人来问题目，有人刻意搭话。马骏说"你现在是名人了"。远舟不适应，还是一个人坐着看书。
- 时代标记：校园里的"名人效应"
- 关键：远舟的"不合群"因成绩被强化

- d: 不适（被关注的不适）
