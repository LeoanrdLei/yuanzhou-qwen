# 1.3 98洪水皖南山区（1998 夏）

**POV**：远舟
**字数预算**：5000
**时间**：1998年夏（暑假）
**地点**：家 / 远郊抗洪现场 / 家
**时代标记**：98大洪水、军民抗洪、新闻联播天天播

## 场景拆分

### 场景 a：家/街上（~1500字）
- 暑假，远舟在家
- 电视里天天播洪水新闻，长江水位上涨
- 街上有人议论，说水要漫过来了
- 远舟骑车出去，看到新安江水位很高
- 空气里潮湿，闷

### 场景 b：远郊抗洪现场（~2000字）
- 父亲单位组织抗洪，去远郊
- 远舟偷偷跟着去了（12岁不让去）
- 看到父亲扛沙袋，沉默、用力
- 父亲发现他，没说话，让他回去
- 远舟没走，站在远处看
- 父亲的衣服湿透了，脸上有泥

### 场景 c：家（~1500字）
- 远舟回到家
- 母亲骂了一顿——"你跑去干嘛！"
- 远舟没吭声
- 母亲又骂——"你爸要是出事怎么办！"
- 远舟低下头
- 晚上，父亲回来了，衣服还是湿的
- 父亲没说话，去洗澡了

## 关键提示
- 本章功能：建立父子关系锚点
- 父亲沉默扛沙袋的形象，为2020 ICU做铺垫
- 远舟偷偷跟去，体现他对父亲的关注
- 母亲骂，体现担心
- 父亲没说话，体现沉默性格
