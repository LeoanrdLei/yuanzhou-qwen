## 二、核心表达（保留原表达）

> 听闻远方有你，难以相信是爱情。
> 人生总有不完美，但那些不完美，本身就是完整的。
> 误会没有完全解开，但都放下了。
> 他们是此生最熟悉、最牵绊的陌生人。
> 远方有你，但身边也有。这就够了。

但**载体**已从言情转为时代史诗：上述表达通过"两个 80 后在 28 年时代洪流中的命运错位"承载，不靠"被截信"等狗血情节硬煽。

### 弱咬合（背景质感）

98 长江洪水（皖南受灾，远舟小学六年级亲历山区灾景）、99 美使馆被炸（黄山一中升旗仪式默哀）、03 非典（若溪高一封校、远舟高二回家被困）、08 奥运/汶川（远舟在上海出租屋看开幕式，第二天得知地震）、12 移动互联网爆发（远舟跳槽方向转移动开发）、19 996.ICU（远舟杭州创业期的争论焦点）、22 上海封城（远舟杭州看着隔壁城市瘫痪、若溪在旧金山看新闻）、23 ChatGPT 浪潮（远舟创业方向被再次洗牌）。

### 删除元素

- ❌ "信被截、22 年后揭谜"（琼瑶式 melodrama）
- ❌ "影楼看到照片"型巧合重逢
- ❌ "物质匮乏"（与林家中产设定冲突，作者已确认不写）

## 五、六卷结构（已锁体量）

### 序章（2026，0.8 万字）

- 远舟在女儿满月散场后阳台喝酒抽烟
- 翻通讯录翻到一个未备注的号码，没点开
- 切回：2002 年冬，黄山下了第一场雪，17 岁的林远舟第一次看见穿紫色羽绒服的女孩

### 卷一 少年得志（1998-2001，4.8 万字）

**主线**：远舟黄山初中称王 / 若溪 POV 进入

- 远舟视角：天赋少年、物理省一、母亲偷加二十块、父亲沉默饭桌、98 洪水皖南山区景象、99 使馆被炸升旗默哀、2001 中考考上屯溪一中
- 若溪视角（新设）：1998 转入黄山某重点小学六年级，父亲调任市交通局副局长（升副后再升正是 2003）；2001 升初一；从小被教"不许跟差生玩"。**两人在 1998-2001 期间在黄山街头/书店有过几次擦肩，互不知道**。
- **卷一高潮**：2001 中考夺魁 + 远舟骑车冲下盘山路，"我无所不能"

### 卷二 紫雪与坍塌（2001-2004，8.8 万字 / 全书最重头）

**主线**：高一/高二 → 紫色外套定情 → 2004 春全线崩塌

- 2001 远舟入屯溪一中，月考第 29 名的冲击，方旭东"你为什么不做最后三题"
- 马骏带入网吧/抽烟/兄弟群
- **2002 冬**：紫色羽绒服初见 → 图书馆借书 → 雪天梧桐树初谈（**爱情萌发小高潮**）
- 2003 非典封校，两人通过窗户递纸条
- 远舟用零花钱买新紫色外套送若溪
- 2003 若溪入屯溪一中高一，与远舟同校；张雨薇在闺蜜视角里多次出现
- 2003 秋徐志远从副局长升任局长，对女儿"差生男友"的清查升级
- **2004 春总崩塌**：父亲下岗→父母离异→远舟堕落性占有式追问→若溪父亲找学校→若溪在走廊里哭→徐志远把女儿转学合肥
- **若溪 POV 章节首次承载情感重量**：她不是被截信、不是误会，她是"被生活按住头按到合肥"，她以为远舟没追是"果然不要我了"
- 2004 高考远舟 398 分
- **卷二末（大坍塌高潮）**：远舟独自骑车上盘山路，下了三天雨，没看到云海

### 卷三 各奔东西（2004-2007，5.6 万字）

**主线**：远舟合肥沉底 / 若溪同城苦读

- 远舟合肥某大专计算机系
- **极致克制的对照**：两人都在合肥，但从未相遇。远舟知道她在中科大，没去找；若溪也知道远舟在合肥某大专，没去找。
- 远舟堕落 → 2006 父亲肝硬化住院 → 远舟楼梯间崩溃哭 → 借室友笔记本第一次学 HTML
- 若溪 POV：合肥一中高三的压抑、中科大外语系的孤独、保送/考研的纠结、与父亲的冷战
- 2007 远舟大专毕业；2007 若溪中科大入学第三年决定考托福
- **卷三末（触底反弹高潮）**：2007 夏，远舟买硬座去上海；同期若溪在中科大图书馆通宵刷 GRE

### 卷四 上海·硅谷（2007-2015，8.8 万字 / 全书第二重头）

**主线**：远舟上海体面公司奋斗 + 2008 金融危机 + 2012 婚礼重逢 + 2015 辞职 / 若溪美研 + H1B + 遇 David

- 2007 远舟到上海，白石洲不存在，改为上海某城中村或共享单间；华强北装机改为上海某外包公司前端
- 2008 金融危机：公司大裁员，远舟被砍 → 转岗 → 触底重学；同期若溪在美西读研第一学期也面对就业冻结
- 2008 奥运 / 汶川：上海出租屋看开幕式、第二天得知地震；若溪在加州看
- 2009 远舟稳定下来，开始做副业接单
- 2010 上海世博，方旭东带学生来上海，远舟招待老师
- 2012 移动互联网爆发，远舟跳槽到一家中型互联网公司
- **2012 第一次重逢**：陈浩宇黄山办婚礼，若溪从美国回来参加，远舟从上海回来。婚礼酒桌上，两人相隔三米看了一眼，对方都端着杯子，谁也没起身。**散场后远舟在老街茶馆静坐了两小时，若溪在父母家阳台抽掉人生第一根烟**。
- 2013-2014 远舟在上海打拼到 28 岁，体面工作，但越做越觉得不是自己想要的
- 若溪 POV：2009 入学 → 2011 毕业找工作 → 2012-2014 跨国公司挣扎拿 H1B → 2015 终于拿到 → 同年通过共同朋友认识 David
- **卷四末（二次出发高潮）**：2015 远舟辞职，赵一鸣劝他留上海，他买了去杭州的高铁票。**同月，若溪与 David 第一次约会在斯坦福校园**。

### 卷五 中年崩塌（2015-2022，7.2 万字 / 全书情感重头）

**主线**：远舟杭州创业、合伙人背叛、父亲病重去世 / 若溪结婚生子、中美关系硬撞 / 2022 黄山相送

- 2015-2017 杭州创业起飞期，孙鹏飞做技术，马骏做后勤
- 2017 春节方旭东撮合远舟与马骏的旧矛盾和解
- 2018 资本寒冬，孙鹏飞与远舟在估值/股权问题上撕破脸，孙带走核心技术团队离开
- 2018 中美贸易战，若溪在硅谷感受到第一波华裔工程师审查
- 2019 远舟父亲肝硬化加重，频繁回黄山
- 2020 父亲去世，远舟在医院 ICU 外站了一夜；同年疫情爆发，公司差点倒闭（背景，弱咬合）
- 2021 远舟独自处理后事、与母亲和远筝的关系变化
- 2021 若溪二女儿出生；同年中美科技战升级，若溪所在公司的中国市场缩减
- 2022 春，徐志远去世，若溪回国办丧事
- **2022 总高潮 / 全书情感顶点**：徐父七天丧期，若溪每天黄山殡仪馆与父母家两头跑。第四天傍晚，她在屯溪老街茶馆遇见远舟（陈浩宇约的，他们彼此都不知道对方在场）。两人各点了一壶毛峰，对坐了一个小时，话不多：
  - "你父亲？"
  - "前年走了。你父亲？"
  - "刚走。"
  - "节哀。"
  - "你也是。"
  - 然后是黄山下了一场雨。
  - 茶凉了，若溪说："那年信寄出去了。但其实……"
  - 远舟说："不用说了。我后来明白了。"
  - 若溪："其实那时候我也不知道自己在想什么。"
  - 远舟："我也不知道。我们都太小。"
  - 散席。她回酒店，他回陈浩宇民宿。
  - 第二天，若溪父亲下葬，远舟没去，他在民宿阳台上听到了远处的鞭炮。
  - 第三天若溪回了上海，远舟送她到机场。车上没怎么说话。
  - **临别她说**："我那件紫色外套，整理我爸遗物的时候才知道——我爸把它收起来了。一直没扔。"
  - **远舟没说话**。
- **2022 章末**：远舟独自开车从机场回杭州，路上想起 2004 年那场雨。

### 卷六 余韵（2022-2026，3.2 万字）

## 八、风格定位（已锁）

- **语言基调**：《繁花》/ 金宇澄南方调——口语高、细节多、场景增厚、"听"得见说话。
- **地域语言层次**：
  - 黄山段：徽州茶馆味，含"毛峰、祁红、徽州话词汇、山区俚语"
  - 合肥段：江淮官话边缘
  - 上海段：金宇澄式上海腔（不是上海话，是上海调）
  - 杭州段：江浙调（融合杭州本地与移民腔）
  - 旧金山段：英语/中式英语混杂，硅谷华人圈调
- **叙事节奏**：
  - 卷一/二慢（青春、黄山、初见、坍塌）
  - 卷三沉（合肥泥潭）
  - 卷四张（上海/硅谷打拼、第一次重逢）
  - 卷五压（创业 + 父丧 + 总高潮）
  - 卷六轻（余韵）
- **总基调**：**不是爽文。不是大团圆。不是言情。是不完美的完整——他们是此生最熟悉、最牵绊的陌生人。爱情存在过，但生活继续了。**
