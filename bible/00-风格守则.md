# 风格守则 V1.0

> 本文件是正文写作的硬性约束。每章起稿前必读，自检时逐条对照。

## 一、总基调

本书是一部**时代史诗**，不是言情、不是爽文、不是大团圆。
核心气质：**《繁花》/金宇澄南方调**——口语高、细节多、场景厚、情绪克制。
读者读完应该觉得：像听一个老朋友讲了一晚上的故事，没哭没笑，但心里堵得慌。

## 二、叙事视角

- **主声部**：林远舟（第三人称有限视角）
- **次声部**：徐若溪（第三人称有限视角）
- 每章只允许一个 POV，**严禁章内切换视角**
- 若溪 POV 章节在章节标题右上角标注 `（若溪）`
- 同一时空的重大事件（如 2004 崩塌、2012 婚礼、2022 相送），**必须双 POV 各写一遍**，留出视差

## 三、语言规范

### 3.1 句式
- 短句为主，长句不超过 40 字
- 多用逗号断句，模拟口语节奏
- 禁止"虽然……但是……"式书面转折
- 禁止排比句超过三组
- 禁止反问句连续使用

### 3.2 用词
- 禁用成语堆砌（"岁月如梭""光阴似箭"等一律删除）
- 禁用网络流行语（"yyds""绝绝子"等，除非角色对话中刻意使用且标注年份）
- 禁用"仿佛""宛如""犹如"连续出现（一章最多 2 次比喻词）
- 形容词能删就删，优先用名词和动词

### 3.3 对话
- 对话不加"他说""她说"——用动作代替（"远舟把烟掐了。'走吧。'"）
- 对话要像真人说话：有停顿、有岔开、有没说完的
- 黄山段对话带徽州口语感（"晓得""蛮好""作死"）
- 上海段带金宇澄调（"不响""……好伐"）
- 不用引号标注内心独白——直接融入叙述

## 四、禁用清单（红线）

以下内容**绝对不写**，写了就删：

| 禁用项 | 原因 |
|--------|------|
| "他感到一阵心痛" | 情绪必须外化为动作/物件 |
| "她的眼眶湿润了" | 同上——写她低头、写她摸杯子、写她看窗外 |
| "命运的安排" | 禁止宿命论叙述 |
| "他终于明白了" | 禁止大彻大悟 |
| "原来这就是爱" | 禁止总结式金句 |
| 任何角色说出书名 | "听闻远方有你"只出现在歌名引用中 |
| 第三者视角全知叙述 | 不写"他们不知道，此刻……" |
| 巧合推动情节 | 禁止"恰好""刚好""冥冥中" |
| 大段景物描写 | 景物不超过 150 字，必须与人物动作绑定 |
| 超过 3 行的省略号 | …… 最多连续 3 个 |

## 五、含蓄度规则

**核心原则**：情绪不直说，用物件/动作/环境外化。

| 要表达的情绪 | 正确写法 | 错误写法 |
|-------------|---------|---------|
| 思念 | 远舟把手机翻过来，屏幕黑了，映出他的脸 | 远舟想念若溪 |
| 心痛 | 他蹲在走廊里，打火机打了三次没着 | 他心痛得无法呼吸 |
| 释然 | 她把那件紫色外套叠好，放进箱子最底层 | 她终于放下了 |
| 孤独 | 食堂里他一个人坐最角落，筷子敲碗边 | 他很孤独 |

## 六、时代细节规则

- 每个场景必须至少有 **2 个可查证的时代标记物**（歌曲/手机型号/价格/新闻事件/流行词）
- 流行歌发行年份必须准确（《那些花儿》2003、《七里香》2004、《南山南》2014）
- 手机型号必须对得上时间线（诺基亚 3310=2000、N73=2006、iPhone 4=2010、iPhone X=2017）
- 价格必须合理（1998 黄山房价 800/㎡、2007 上海合租 1500/月、2015 杭州房价 20000/㎡）
- 不确定就查，查不到就模糊处理（"那年的房价还没涨到天上去"）

## 七、章节格式

```markdown
# 第 X 章 章节名

（正文，纯中文，无 markdown 格式标记）
```

- 章节标题格式：`# 第 X 章 时间关键词`
- 正文不加粗、不斜体、不用 markdown 格式
- 场景切换用空行分隔，不用分割线
- 每章开头不写"时间/地点"标签——融入叙述

## 八、字数控制

| 卷 | 单章预算 | 弹性范围 |
|----|---------|---------|
| 序章 | 8000 | 6000-10000 |
| 卷一/三 | 5000-6000 | 4000-7500 |
| 卷二/四 | 6000-7000 | 5000-8500 |
| 卷五 | 6000-8000 | 5000-10000（父丧章可到 10000） |
| 卷六 | 4000-5000 | 3000-6000 |
| 尾声 | 3000 | 2000-4000 |

## 九、自检清单（每章写完后逐条对照）

- [ ] 视角是否统一（本章 POV 是谁？有无越界？）
- [ ] 禁用清单是否触雷（逐条搜索）
- [ ] 情绪是否外化（搜索"感到""觉得""意识到"）
- [ ] 时代细节是否查证（本章年份的标记物够不够？）
- [ ] 意象是否按贯穿表出现（本章该触发哪个意象？）
- [ ] 字数是否在预算范围内（±20%）
- [ ] 对话是否像人话（读出声来试试）
- [ ] 景物是否超过 150 字
- [ ] 有无"作者插话"（"读者也许会觉得……""说到这里不得不提……"）
