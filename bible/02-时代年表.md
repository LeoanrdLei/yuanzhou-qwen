# 时代年表 V1.0

---

## 咬合策略

**中等咬合**：不逐年堆砌新闻，只锁定 4 个强咬合节点直接改变主角命运。其余年份用"背景音"方式渗透（1-2 个标记物即可）。

**四个强咬合节点**：
1. **2004** 国企改制 → 林建国下岗 → 父母离异 → 若溪被迫转学
2. **2008** 金融危机 → 远舟被裁 → 若溪留学犹豫
3. **2015** 互联网+ → 远舟辞职去杭州创业 → 若溪 H1B 抽签
4. **2018-2020** 资本寒冬+中美关系+疫情 → 合伙人背叛 → 父丧 → 若溪无法回国
