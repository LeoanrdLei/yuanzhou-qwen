## 一、黄山（安徽）

**定位**：故事起点，情感原乡。小城、慢、山水气。

### 关键地点

| 地点 | 说明 | 出现章节 |
|------|------|---------|
| 远舟家 | 老城区，父亲单位宿舍楼，三层，楼道暗 | 1.1, 2.x |
| 初中 | 虚构校名"黄山市第一中学"（初中部），城郊 | 1.1-1.3 |
| 书店 | 老街上的新华书店，远舟与若溪第一次说话的地方 | 1.4 |
| 老街/屯溪老街 | 青石板路，茶馆，徽州建筑 | 散见 |
| 盘山路 | 中考后远舟骑车冲下的山路（卷一末高潮） | 1.8 |
| 化肥厂 | 父亲工作单位，城郊，大门朝西，公告栏在门口 | 1.1, 2.x |
| 光明顶/云海 | 黄山标志性景观，全书核心意象载体 | 序章, 卷二末, 尾声 |
| 若溪家 | 新城区，父亲单位分的房，比远舟家好 | 1.7 |

### 氛围关键词
- 青石板、白墙黛瓦、马头墙
- 潮湿、多雨、春天梅雨
- 茶馆、毛峰茶
- 小城人情社会——谁认识谁

## 二、合肥（安徽）

**定位**：远舟的沉底期（2004-2007），若溪的高中+大学。两人同城不同命。

### 关键地点

| 地点 | 说明 | 出现章节 |
|------|------|---------|
| 合肥一中 | 若溪转学后的高中（2004-2007） | 2.x, 3.x |
| 中科大 | 远舟的大学（2005-2009），若溪后来也入读（2007-2011） | 卷三 |
| 远舟出租屋 | 合肥某城中村，便宜、潮湿 | 3.x |
| 医院 | 2006 父亲住院的地方 | 3.x |
| 某街道 | 两人同在合肥却擦身而过的街道（极致克制章） | 3.x |

### 氛围关键词
- 灰、平、不南不北的城市
- 中科大的学术气 vs 城中村的市井气
- 冬天冷、夏天热
